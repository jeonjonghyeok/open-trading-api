이 시스템이 하는 일

기존에 사람이 직접 하던 3가지 일을 AI가 대신합니다.

📊

시장 분석

주가, 거래량, 추세를 실시간으로 파악하고 매수/매도 시점을 판단합니다. 사람이 차트를 볼 필요가 없습니다.

⚡

자동 주문 실행

판단이 내려지면 한국투자증권 API를 통해 즉시 주문을 넣습니다. 사람이 매매 버튼을 누를 필요가 없습니다.

🧠

전략 자동 개선

거래 결과를 분석해 잘 통하는 전략은 더 강화하고, 성과가 부진한 전략은 자동으로 수정하거나 교체합니다.

자동화 구조

3가지 자동 루틴

시스템은 3개의 Claude Code Scheduled Task가 시간대별로 맞물려 돌아갑니다. 모두 사람의 개입 없이 실행됩니다.

루틴 1 · kis-data-collect
캐시 워밍
⏰ 평일 08:00 (장 시작 1시간 전)
OHLCV·후보 종목 데이터를 미리 가져와 캐시합니다. 09:00 장 시작 즉시 캐시 히트로 빠르게 첫 사이클을 돌립니다.
루틴 2 · kis-trading-cycle
장중 매매
⏰ 평일 09:00~15:00, 15분마다
RSI·MACD 신호를 확인하고, 조건이 맞으면 KIS 모의투자 API로 즉시 주문을 실행합니다. 결과는 Notion에 실시간 기록됩니다.
루틴 3 · kis-daily-recap
복기 + 전략 진화
⏰ 평일 15:35 (장 마감 5분 후)
오늘 성과를 분석하고 AI가 전략 파라미터를 자동 조정합니다. 복기 결과는 Notion 전략 히스토리 DB에 자동 기록됩니다.

기대 효과

이 시스템이 만드는 변화

자동화를 통해 얻을 수 있는 실질적인 이점들입니다.

⏱️
시간 해방
차트를 매일 보지 않아도 됩니다. 시스템이 시장을 지켜보는 동안 다른 일을 할 수 있습니다.
😤
감정 제거
공포와 탐욕 없이 규칙대로만 거래합니다. 충동적인 매수·매도를 원천 차단합니다.
📈
전략 진화
시장이 바뀌면 전략도 바뀝니다. 과거 성과 데이터를 기반으로 AI가 지속적으로 개선합니다.
🔬
다전략 실험
여러 전략을 동시에 소규모로 실험해 성과가 좋은 것을 자동으로 확대 적용합니다.
